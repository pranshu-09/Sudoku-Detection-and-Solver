# Sudoku-Detection-and-Solver

Through this project, we are able to Detect and Predict digits from an Unsolved Sudoku Image using Deep Learning and solve the Sudoku using Backtracking Algorithm.

## Libraries Used:
* OpenCV
* NumPy
* OS
* Scikit-Learn
* TensorFlow
* Pillow
* Matplotlib
* Tabulate
